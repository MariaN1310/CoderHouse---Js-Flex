App Lista de Compras y Lista de Tareas

La idea de esta aplicación es ayudarte a organizar dos tipos principales de listas: una lista de compras y una lista de tareas por hacer. Permite al usuario agregar productos y especificar cantidades para su lista de compras, marcando los productos como comprados al tenerlos. También permite añadir tareas pendientes a la lista de cosas por hacer, marcándolas como completadas a medida que se realizan.
